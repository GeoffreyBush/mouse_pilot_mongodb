How to edit the details of an existing breeding cage

To edit an existing breeding cage, navigate to the breeding cage page:

Look for the breeding cage you want to edit, and locate the pencil icon:

Click the icon to open the edit cage page for that particular breeding cage.

On the edit cage page, you should be able to see all current data that are stored in the database:

Modify the data you wish to change and Click Save Changes button to proceed.

The database will be updated with the data you've just inserted.
