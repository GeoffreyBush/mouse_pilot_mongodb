How to edit the details of an existing mouse

To edit an existing mouse, make sure you are on the correct Project page:

Look for the mouse you want to edit, and locate the pencil icon:

Click the icon to open the edit mouse page for that particular mouse.

On the edit mouse page, you should be able to see all current data that are stored in the database:

Modify the data you wish to change and Click Save Changes button to proceed.

The database will be updated with the data you've just inserted.
