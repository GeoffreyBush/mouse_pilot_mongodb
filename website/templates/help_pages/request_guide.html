How to add a new request to one (or more) mouse

To add a request, make sure you are on the correct Project page:

Tick the checkbox of the mouse you want to include in the request. You can also tick multiple mice.

Click the Add Request button to open the add request page:

The mice chosen will appear here. You can proceed to insert other details for the request.

You can also modify which mouse to choose during this phase.

Click the Add Task button to confirm adding the request.

To check the current requests related to your account, navigate to the Request page by clicking in the navigation bar:

After the request is completed, you can click the Confirm Request button to finish the request. After finishing the request, the modified data will automatically be updated in the database.

During the request, you can also see which mouse is involved by checking the highlighted row in the project view:
