How to use the filter to view certain data

At the top of every project view, there is a filter. The filter consists of several text inputs and selections which correspond to the table columns:

Insert a particular data of that column and click the Search button, the database will filter out all mouse instances that do not contain that particular data.

You can filter the table with multiple column at once.

To reset the filter, simply click the Cancel button.
